# GameSphere 🎮

GameSphere is a full-stack web application that helps users discover games and check if their system can run them.

## 🚀 Features

* 🎯 Game discovery
* 📄 Game details
* 💻 Compatibility checker
* 🤖 Recommendations

## 🛠️ Tech Stack

* Frontend: React
* Backend: FastAPI
* Database: MongoDB
* API: RAWG API

## ⚙️ How to Run

### Backend

cd backend
python -m uvicorn server:app --reload

### Frontend

cd frontend
npm install
npm start

## 📌 Architecture

User → Frontend → Backend → API/Database
